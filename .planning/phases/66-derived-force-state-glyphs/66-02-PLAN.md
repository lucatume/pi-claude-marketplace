<objective>
Implement THE DERIVER (D-66-01/02): in `orchestrators/plugin/list.ts::installedRowMessage`,
derive the realized status of a recorded-installed plugin from (persisted `compatibility`
record + no-network candidate resolve). force-installed is read from the EXISTING persisted
`record.compatibility` (no new flag, no migration); force-upgradable splits the existing
upgradable branch using a no-network `resolveStrict` of the candidate manifest entry.

Purpose: This is the single shared derivation that all list rows consume. FSTAT-03
(auto-return to installed after a supported upgrade) falls out for free because upgrade
rewrites `compatibility` and the deriver reads it live.
Output: The 4-way predicate (force-installed -> force-upgradable -> upgradable -> installed)
in installedRowMessage, plus a unit matrix proving derivation purity, ordering, and
no-network behavior.

Depends on 66-01 for the `force-installed`/`force-upgradable` status union arms and render
arms. Runs in Wave 2 parallel with 66-03 and 66-04 (disjoint files).
</objective>
