<objective>
Thread the derived force signal into the detail and success surfaces (D-66-04 / FSTAT-07):
`/claude:plugin info` reports `force-installed` and surfaces the dropped-component detail
(via the shared `narrowUnsupportedKinds` marker helper) for an installed plugin that
re-resolves `unsupported`; and the `install --force` / `update --force` SUCCESS cascade row
reads `force-installed` (not `installed`) when the live resolved state is `unsupported`.

Purpose: A force install/update must report its true realized state on every surface the
user sees — info detail and the success notification — sourced from the same derived
signal as the list deriver (D-66-01's single-deriver consistency).
Output: info force-installed + dropped-component detail; install/update force-installed
success rows; unit coverage on all three orchestrators.

Depends on 66-01 for the `force-installed` status arm + info-row glyph. Runs in Wave 2
parallel with 66-02 (list.ts) and 66-04 (reconcile/notify.ts) — disjoint files.
</objective>
