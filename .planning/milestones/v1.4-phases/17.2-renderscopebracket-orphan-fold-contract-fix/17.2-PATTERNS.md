function renderScopeBracket(scope: Scope | undefined): string {

```typescript
function renderScopeBracket(pluginScope: Scope | undefined, mpScope: Scope): string {

  return `[${pluginScope}]`;
}

Replace the line "the bracket renders `""` when scope is undefined" with "the bracket renders `""` when `pluginScope === undefined` OR `pluginScope === mpScope` (orphan-fold contract per D-16-17 + SNM-11)".
