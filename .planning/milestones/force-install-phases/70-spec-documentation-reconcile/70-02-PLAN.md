    `force-upgradable` (a clean installed plugin whose newer no-network candidate would
    NEWLY degrade it, glyph `●`); a force-installed plugin is never force-upgradable,

    SEV-02 severity split (no-`--force` `unsupported` install = error WITH the hint;
    structural `unavailable` install = error WITHOUT the hint).
    (d) D-70-03 / WR-01: document that the missing-soft-dep-companion warning is scoped
    to `install` and manual `update` success ONLY and is intentionally NOT added to the
    marketplace autoupdate cascade (autoupdate severity is governed by SEV-03; its
