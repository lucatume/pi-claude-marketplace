<objective>
Rename the **user-visible output vocabulary** and the **persisted completion-cache literals** from
`force`/`unsupported` to the `partial`/`partially-available` vocabulary — atomically, so the
byte-equality catalog UAT never goes RED. This is the byte-gated half of Phase 75 (D-75-01, locked
render forms in 75-CONTEXT.md). Depends on Plan 01 (flag + degrade plumbing already renamed).

**Byte-safety of the task seam (read carefully — this is NOT a forbidden split):** the three tasks
split along a byte-SAFE axis, NOT along the forbidden verdict-vs-force-state axis.
- **Task 1** renames the *identifiers/literals* (status literals, message-interface names, glyph
  const NAMES, resolver state, classifier, cache literals, `status:`/`state:` comparisons, fixture
  `status:` fields). It keeps every user-visible `(…)` render string and hint-trailer body OLD, so
  `notify()` output is byte-identical → the catalog UAT stays GREEN (a `case "partially-available":`
  arm that still `return`s `"(unsupported)"` renders the same bytes). **Verdict AND force-state
  literals move together in this one task.**
- **Task 2** flips the *user-visible strings* — every `(…)` render string, both hint-trailer bodies,
  the two hint-trailer const NAMES — in lockstep with `docs/output-catalog.md` and the
  `catalog-uat` FIXTURE expected bytes, in ONE commit (the atomic-supersession triad). **Verdict AND
  force-state render strings move together in this one task.**
- **Task 3** adds the grep absence/presence guard + CHANGELOG.

Neither task ever splits the verdict rename from the force-state rename; the literal-vs-render-string
seam renders identically at the Task-1 boundary, so there is no RED byte window.

Apply the RESEARCH §4c symbol-level rule throughout — the in-scope standalone `"unsupported"`
literal co-occurs on the same lines as the OUT-of-scope `"unsupported source"` / `.unsupported[]` /
`unsupportedKinds`. **No blind substring replace.**

Purpose: deliver the locked partial vocabulary on every user-visible surface with the byte gate as
the safety net.
Output: the partial render tokens/status literals, `ICON_PARTIALLY_*`, `PARTIAL_*_HINT_TRAILER`,
`partialHint`, the renamed message interfaces + resolver verdict, completion-cache schemaVersion 4,
the reconciled docs, and the new surgical-completeness guard.
</objective>
