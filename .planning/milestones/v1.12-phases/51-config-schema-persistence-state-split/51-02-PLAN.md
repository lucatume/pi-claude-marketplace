ownership-split is structurally enforced. Plan 03 then locks ownership at the architecture

`persistence/migrate.ts`, extended tests, and a new legacy fixture
`tests/persistence/fixtures/legacy/state-with-autoupdate.json`.

unknown keys + write-back preservation contract for downstream Phase 56), D-12 (no
STATE_SCHEMA bump; `autoupdate` carve-out), D-13 (ORDERING RAIL: scrub gated by
