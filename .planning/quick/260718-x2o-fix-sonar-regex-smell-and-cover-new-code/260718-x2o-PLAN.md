**files:** extensions/pi-claude-marketplace/orchestrators/plugin/clone-cache.ts

**action:** In `readOriginRemoteUrl`, replace `/^url\s*=\s*(.+)$/` with a
linear-time equivalent. Recommended: `/^url\s*=(.*)$/` + trim the capture and
return it only when non-empty (the input `line` is already `.trim()`ed, so
behavior is identical: a bare `url =` still yields undefined). The ambiguity
Sonar flags is `\s*` and `(.+)` both matching spaces; `=(.*)` removes the
overlap. Do NOT change the `[remote "origin"]` section regex (not flagged).
